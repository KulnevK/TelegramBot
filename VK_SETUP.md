# Настройка VK бота

## 1. Создание сообщества ВКонтакте

1. Зайдите на https://vk.com
2. Нажмите **Сообщества** → **Создать сообщество**
3. Выберите тип **Бизнес** или **Тематическое сообщество**
4. Заполните название и описание
5. Создайте сообщество

## 2. Включение сообщений

1. Перейдите в **Управление сообществом**
2. Откройте **Сообщения** → **Настройки для бота**
3. Включите **Возможности ботов**
4. Включите **Разрешить добавление сообщества в беседы**

## 3. Получение токена доступа

1. Перейдите в **Управление** → **Работа с API**
2. Нажмите **Создать ключ**
3. Выберите права:
   - ✅ Управление сообществом
   - ✅ Сообщения сообщества
   - ✅ Документы
4. Скопируйте токен (он начинается с `vk1.a...`)

## 4. Настройка Long Poll API

1. Перейдите в **Управление** → **Работа с API** → **Long Poll API**
2. Включите **Long Poll API**
3. Выберите **Версию API**: 5.131 или выше
4. Включите типы событий:
   - ✅ Входящие сообщения

## 5. Добавление токена в Railway

1. Зайдите на https://railway.app
2. Откройте ваш проект с ботом
3. Перейдите в **Variables**
4. Добавьте новую переменную:
   - **Name:** `VK_TOKEN`
   - **Value:** ваш токен из шага 3
5. Сохраните

## 6. Проверка работы

После деплоя:
1. Откройте ваше сообщество ВКонтакте
2. Напишите сообщение боту: `/start`
3. Бот должен ответить приветствием
4. Отправьте ссылку на видео или музыку

## Примечания

- Бот работает одновременно и в Telegram, и в VK
- Если не установлен `VK_TOKEN`, будет работать только Telegram бот
- Если не установлен `BOT_TOKEN`, будет работать только VK бот
- Можно запускать оба бота одновременно

## Локальный запуск

Для локального тестирования:

```bash
# Установите зависимости
pip install -r requirements.txt

# Установите переменные окружения
export BOT_TOKEN="ваш_telegram_токен"
export VK_TOKEN="ваш_vk_токен"
export YANDEX_MUSIC_TOKEN="ваш_яндекс_токен"

# Запустите оба бота
python run_bots.py

# Или запустите только один бот
python bot.py        # только Telegram
python vk_bot.py     # только VK
```

## Ограничения VK

- Максимальный размер файла: 50 МБ (как в Telegram)
- VK API имеет лимиты на количество запросов
- Документы отправляются как вложения, а не как встроенные медиа
