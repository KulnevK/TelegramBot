# Функция скачивания музыки

## Что добавлено

Бот теперь поддерживает скачивание музыки с популярных музыкальных сервисов:

- **Яндекс.Музыка** - через библиотеку `yandex-music`
- **Spotify, SoundCloud, Bandcamp, Deezer** - через `yt-dlp`

## Как использовать

Просто отправьте боту ссылку на трек:

```
https://music.yandex.ru/album/12345/track/67890
https://open.spotify.com/track/xxxxx
https://soundcloud.com/artist/track
```

Бот автоматически определит тип контента (видео или музыка) и скачает в нужном формате.

### Формат скачивания
- Музыка скачивается в формате MP3 (192 kbps)
- Имя файла: `Исполнитель - Название.mp3`
- Лимит размера: 50 МБ (ограничение Telegram)

## Ограничения

1. **Яндекс.Музыка**: работает без авторизации, но некоторые треки могут быть недоступны
2. **Spotify**: требуется активная подписка на сервисе для доступа к некоторым трекам
3. **Общий лимит**: 3 скачивания/день для бесплатных пользователей (включая видео и музыку)
