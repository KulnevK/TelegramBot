# Telegram Video & Music Downloader Bot

Бот для скачивания видео и музыки из социальных сетей и музыкальных сервисов.

## Возможности

### 🎥 Видео
- YouTube
- TikTok
- Instagram
- Twitter/X
- Facebook

### 🎵 Музыка
- Яндекс.Музыка
- SoundCloud
- Bandcamp

### Дополнительно
- Система лимитов (3 скачивания/день для бесплатных пользователей)
- Премиум подписка через ЮKassa
- База данных пользователей
- Статистика скачиваний

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте переменные окружения (см. ниже)
4. Запустите: `python bot.py`

## Переменные окружения

- `BOT_TOKEN` - токен Telegram бота от BotFather
- `YANDEX_MUSIC_TOKEN` - токен Яндекс.Музыки для скачивания треков
- `ADMIN_IDS` - ID администраторов через запятую (например: "123456789,987654321")
- `YOOKASSA_SHOP_ID` - ID магазина ЮKassa (опционально)
- `YOOKASSA_SECRET_KEY` - секретный ключ ЮKassa (опционально)

## Деплой на Railway

1. Форкните этот репозиторий
2. Создайте проект на Railway.app
3. Подключите GitHub репозиторий
4. Добавьте переменные окружения в настройках Railway:
   - `BOT_TOKEN` - для Telegram бота
   - `YANDEX_MUSIC_TOKEN` - для скачивания с Яндекс.Музыки
5. Деплой произойдет автоматически
